Logical operators:
Set of symbols that we can use to combine multiple boolean values.
There are logical operators ! ||| &&

Why should me learn about logical operators?
We can greatly simplify the logic if we use these operators

What is the syntax?
They only work with boolean data type
if(! condition)
if(condition1 || condition 2)
if(condition1 || condition 2)

Not operator "!"
Not is used to negate the boolean values
